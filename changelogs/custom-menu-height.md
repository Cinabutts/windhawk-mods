## 1.1 ([Jul 2, 2026](https://github.com/ramensoftware/windhawk-mods/blob/9dc9145d6cb2577c40c716476cc086d3231c9ab1/mods/custom-menu-height.wh.cpp))

* Updated the context menu item height setting to handle specific values for default states:
  * `0`: Uses the default system value and keeps immersive menus.
  * `-1`: Uses the default system value and eradicates immersive menus system-wide.
* Added support for eradicating the immersive menu from the Windows Update system tray icon on Windows 10.
* Improved OS version detection required to eradicate immersive menus system-wide on Windows 10 and later.
